# Project Handover Sheet - Standard Format

Use this same structure for every job so handovers are consistent. The Streamlit
app produces this exact layout, but here it is as a plain template you can also
fill in by hand if needed.

---

## 1. Project Overview

- Project name:
- Project number:
- Client / builder:
- Site address:
- Status (eg 65%, install underway):
- Contract ref:
- Outgoing PM:
- Incoming PM:
- Handover date:

## 2. Handover Summary

Plain-language paragraph. Where the job is at, the main risks, and what the
incoming PM needs to action first. Keep it short.
